# ⏱ Pomodoro Timer Chrome Extension

A simple and lightweight Chrome extension to boost productivity using the Pomodoro technique.

## 🚀 Features
* Start / Pause timer
* Clean popup UI
* Easy to use

## 🛠 Tech Stack
* HTML
* CSS
* JavaScript
* Chrome Extension API

## 📦 Installation
1. Download or clone this repo
2. Open Chrome and go to:
   chrome://extensions/
3. Enable Developer Mode
4. Click "Load unpacked"
5. Select this project folder
